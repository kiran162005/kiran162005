> **Core Pipeline:** *Python | Streamlit | Faster-Whisper | Gemini Pro API*

* Built contextual abstraction prompts inside Gemini to yield logical summary nodes, multi-variant flashcards, and automated interactive quiz modules.
* Optimized multi-format file exporters running concurrent compression into single structured target downloads (`.txt`, `.pdf`, `.zip`).

### 🏫 2. SmartClass – Collaborative Virtual Classroom
> **Core Pipeline:** *MongoDB | Express.js | React.js | Node.js | Socket.io | Groq API*

* Engineered a collaborative browser canvas utilizing HTML5 APIs to map multi-user teaching streams smoothly.
* Intercepted ongoing lesson materials with an asynchronous pipeline linked to Groq API to extract live quiz evaluations and notes instantly.
