## 🚀 Projects

* 🎤 **AI Voice-to-Notes Generator** – *(Python, Streamlit, Faster-Whisper, Gemini API)*
  * Built an end-to-end processing pipeline to transcribe, summarize, and convert lecture audio files into comprehensive revision material.
  * Integrated Faster-Whisper for low-latency speech recognition alongside Gemini LLM for smart contextual quiz and flashcard generation.
  * Implemented file exporting mechanisms to package multi-format reports (TXT, PDF, ZIP) with live interactive quiz evaluation logic.

* 🏫 **SmartClass – Collaborative Virtual Classroom** – *(MERN Stack, Socket.io, Groq API)*
  * Architected a full-stack virtual classroom featuring real-time peer collaboration using Socket.io.
  * Engineered an interactive, live teaching canvas paired with instant real-time messaging chat components.
  * Embedded an LLM-powered assistant utilizing Groq API for automated live quiz and study-note generation.
  * Designed modular REST APIs and automated text extraction workflows for PDF/DOCX document processing.

* 📊 **GitGrade – GitHub Repo Evaluator** – *(Python, Streamlit, GitHub REST API)*
  * Developed a metrics analyzer assessing repository source code using GitHub REST API data across structural depth, documentation, testing, and commit velocity.
  * Formulated an automated grading engine (0–100 scale) providing customized engineering feedback and structural improvement roadmaps within a Streamlit UI.

* ♿ **Accessibility Analyzer** – *(Web Auditing Tool)*

  * Implemented automated reporting mechanics to compile findings into clean, exportable PDF audit logs.
